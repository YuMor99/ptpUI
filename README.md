# ptpUI
## 页面一
- 活动：MainActicity
- 布局：activity_main.xml
- 控件id：
  - `toolbar`：左边是个滑动菜单
  - `vitamio_videoView`：用Vitamio库获取视频流
  - `btn_take_photo`：拍照
  - `edit`：输入视频流地址
  - `photo`：查看相册

## 页面二
- 活动：AlbumActivity
- 布局：activity_album.xml, layout_photos.xml
- 控件id：
  - `photo_image`：照片
  - `photo_name`：暂时只放照片名，后续可改为照片信息，或点击照片查看详细信息
